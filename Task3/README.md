# Heart Disease Prediction

## Objective
Predict heart disease based on patient clinical data using Logistic Regression.

## Dataset
Heart Disease UCI dataset with numeric, boolean, and categorical features.

## Model Used
- Logistic Regression

## Evaluation
- Accuracy
- Confusion Matrix
- ROC Curve & AUC

## Results
- Top features influencing heart disease identified
- ROC curve shows strong discriminative performance
- Dataset preprocessing and scaling improved model accuracy
