# Stock Prices Prediction

## Objective
Predict next day stock prices using historical data with Linear Regression.

## Dataset
- Apple Inc. (AAPL) historical stock data (Jan 2022 - Jan 2024)
- Features: Open, High, Low, Volume
- Target: Next day Close price

## Model Applied
- Linear Regression

## Evaluation
- Mean Squared Error (MSE)

## Results
- Linear Regression captures general price trend
- Visualization shows predicted vs actual prices
- Financial stock data is volatile; more advanced models may improve accuracy
