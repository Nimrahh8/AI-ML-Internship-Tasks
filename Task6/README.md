# House Price Prediction

## Objective
Predict house prices using machine learning based on property features.

## Dataset
- CSV dataset with features: SquareFeet, Bedrooms, Bathrooms, YearBuilt, Neighborhood, Price

## Model Used
- Linear Regression

## Evaluation
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Results
- Model captures the relationship between features and house price.
- Scatter plot shows predicted prices follow actual prices.
- Square footage, number of bedrooms, and bathrooms positively influence price.
